# Quickstart: Profile Tab UI

This is a UI-only feature that displays a new Profile Tab and its sub-screens.

## To test and view the screens:
1. Open the app and navigate to the **Profile** tab in the bottom navigation bar.
2. **Main Profile**: View the user info, wallet card, completion progress, and settings list.
3. **QR Code Screen**: Tap the QR Code icon button in the top right.
4. **Profile Info (Edit)**: Tap the pencil edit icon next to the user name.
5. **Appearance**: Tap the "Appearance" list tile to select themes, colors, and backgrounds.
6. **Chat Theme Preview**: From the Appearance screen, tap the "Preview Chat" button at the bottom.

All data is mocked and all strings are localized. No real API calls are made.
