# Quickstart: Chat Optimization

## Running the Optimized Chat
1. Launch the application: `flutter run`
2. Navigate to the chat view with another user.
3. Test rapid typing indicators; observe smooth performance with no whole-page UI stutters.
4. Send a rapid succession of messages; verify smooth scrolling.

## Testing Call Overlay
1. While typing a message in the text chat, initiate an incoming call from another device.
2. Verify the call UI appears as a non-blocking overlay.
3. Accept the call and verify you can continue typing the text message without losing state.
